Application d'essayage de lunettes en ligne.

l'application se devise en 2 partis :

coté adminstrateur et coté client
cote client se compose d'un catalogue de lunette impoter de jeeliz un api open source deployer sur github, il permet de faire un filtre de lunette pour les essayer ansi qu'un button de d'achat pour valider les commandes.

cote administrateur nous donnes la possibilités d'ajouter des articles ,modifier les prix, ou supprimer un article,
ainsi que valider ou annuler une transaction .

l'application est vraiment tres basic elle peut etre utiliser par des gens de differente age commencent par 12ans.
